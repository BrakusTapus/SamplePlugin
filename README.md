## Installation

* Copy the Custom Repo Link
```
https://raw.githubusercontent.com/Brakus-Tapus/MyDalamudPlugins/main/pluginmaster.json
```
* Open dalamud settings in game
* Paste the link into the Custom Plugin repositories found in the "experimental" tab.
* Pres the 'Save & Close' or 'Save' button 
* Install as normal from Dalamud Plugins window.
